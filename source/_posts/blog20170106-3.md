## jsp配置
```
<servlet>
	<servlet-name>configuration</servlet-name>
	<jsp-file>/configuration.jsp</jsp-file>
	<init-param>
		<param-name>name</param-name>
		<param-value>xiaomei</param-value>
	</init-param>
</servlet>
<servlet-mapping>
	<servlet-name>configuration</servlet-name>
	<servlet-name>/configuration</servlet-name>
</servlet-mapping>
```

## jsp-config 标签
```
<jsp-config>
	<taglib>
		<taglib-uri></taglib-uri>
		<taglib-location></taglib-location>
	</taglib>
	<jsp-property-group>
		<url-pattern></url-pattern>
	</jsp-property-group>
</jsp-config>
```
