三种dom加载方式
```
//第一种
$(function(){
	
})

//第二种
$(document).ready(function(){
	
})

//第三种
$(document).on("ready",function(){
	
})
```

等加载外部js完成后，再执行$(function(){})
```
$.holdReady(true);

$.getScript("test.js",function(){
	$.holdReady(false)
});

$(function(){
	//这样等test.js加载完成执行
})


```
