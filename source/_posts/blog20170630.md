## 设置淘宝镜像方法：

在nodejs\node_modules\npm\npmrc文件中添加 :
```
registry=https://registry.npm.taobao.org
```

## npm https协议不支持 期待http？

err message：
```
npm ERR! Protocol "https:" not supported. Expected "http:"
```
solution：

```
npm config rm proxy
npm config rm https-proxy
```
